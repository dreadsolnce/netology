# GitLab
#### платформа для управления проектами и репозиториями исходного кода.
___

- В её основе лежит система контроля версий Git
- Платформа доступна как облачный сервис
- Хранение кода (репозиторий)
- Планирование и трекинг задач
- CI/CD-пайплайны — автоматизация сборки
- Контроль доступа — гибкая система ролей
- Управление релизами
- Хорошая документация
- Мониторинг и безопасность
- Позволяет запуск сборок по событиям
- Имеются возможности безопасного хранения секретных данных
- Позволяет разворачивать агентов сборки на собственных серверах

Это одна из самых популярных систем, а так же позволяет развернуть систему на собственных производственных мощностях либо в облаках.
